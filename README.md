# AI3000R - Artificial Intelligence for Business Applications

This repository contains coursework and assignments for **AI3000R - Artificial Intelligence for Business Applications** at the University of South-Eastern Norway (USN).

## Course Overview

This course focuses on the impact of Artificial Intelligence on organizations and society. Students learn principles and methods for developing smart AI data-dependent solutions through machine learning, deep learning, and data mining techniques applied to practical industry challenges.

## Repository Structure
```
├── assignment_01/          # What Is Intelligence? - Philosophical exploration
├── assignment_02/          # Customer Churn Prediction - Classification project
├── assignment_03/          # Sales Forecasting - Regression analysis
└── README.md
```

Each assignment folder contains its own README with detailed instructions and requirements.

## Learning Objectives

- Understanding AI principles and methods
- Implementing machine learning solutions to real-world business problems
- Analyzing data through various AI techniques
- Developing practical AI-driven projects
- Ethical and legal considerations in AI applications

## Technologies

- Python 3.x
- scikit-learn
- NumPy
- Matplotlib
- TensorFlow 2
